# Web приложение определения заполненных форм.

Web приложение "SecretLook" предназначено для проведения опросов и анонимного анкетирования.
Позволяет зарегистрированному клиенту в личном кабинете создать опрос, анкеты и разместить
ссылку на них у себя. Клиент может получать статистику по итогам опроса и анкетирования.
(front реализован для оценки функциональности)

## Установка и запуск
Выполнить следующие команды:

- Клонировать репозиторий: `git clone https://github.com/Kandrey1/Test_FormCompleted.git`

### Без Docker
- Создать виртуальное окружение командой: `python -m venv venv`

- Установить пакеты: `pip install -r requirements.txt`

- В файле config.py установить `MONGODB_HOST = 'localhost'`

У Вас должнa быть установлена база данных MongoDb на порту 27017. Если она на 
другом порте тогда в файле config.py установить нужный порт `MONGODB_PORT = `

- Для запуска приложения выполнить команду `python main.py`


### Используя Docker
- В файле config.py установить `MONGODB_HOST = 'mongo'`

 - Находясь в корне проекта выполнить: 

`docker-compose up --build -d`


## Тестирование
### Функций
Для тестирования необходимо уставить pytest `pip install pytest`
Выполнить команды для запуска тестирования `pytest tests`

## Тестирование запросов (главная функция приложения)

Для создания начальных данных, после запуска приложения (и БД если через докер)
перейти по адресу `http://127.0.0.1:5000/test_data/create`


Используя 'postman', 'insomnia' или другую аналогичную программу после запуска 
приложения, ее можно протестировать запросами из файла `script_for_test.py`
или перейти по адресу `http://127.0.0.1:5000/script_test/start`.
